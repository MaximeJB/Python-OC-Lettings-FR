Technologies utilisées
======================

Backend
-------

* **Django 5.0** : Framework web Python
* **Python 3.12** : Langage de programmation
* **SQLite** : Base de données (développement)
* **Gunicorn** : Serveur WSGI pour la production

Tests et qualité du code
-------------------------

* **pytest** : Framework de tests
* **pytest-django** : Plugin pytest pour Django
* **pytest-cov** : Couverture de code
* **flake8** : Linter Python (PEP8)

Monitoring et logging
---------------------

* **Sentry** : Surveillance des erreurs en production
* **Python logging** : Système de logs

DevOps et déploiement
----------------------

* **Docker** : Conteneurisation
* **GitHub Actions** : CI/CD
* **Render** : Hébergement cloud
* **Docker Hub** : Registry d'images Docker

Documentation
-------------

* **Sphinx** : Générateur de documentation
* **Read The Docs** : Hébergement de la documentation
* **reStructuredText** : Format de documentation
